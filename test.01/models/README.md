# 3D模型目錄

請將您的GLTF格式3D模型文件放在此目錄中。

## 推薦資源

- [Sketchfab](https://sketchfab.com/) - 免費3D模型下載
- [Poly Haven](https://polyhaven.com/models) - 免費3D模型
- [Free3D](https://free3d.com/) - 免費3D模型庫

## 格式要求

- 支持格式：`.glb` 或 `.gltf`
- 建議使用 `.glb` 格式（單文件，載入更快）
- 模型大小建議控制在 5MB 以內

## 示例

將模型文件命名為 `robot.glb` 並放在此目錄，即可在AR場景中使用。
